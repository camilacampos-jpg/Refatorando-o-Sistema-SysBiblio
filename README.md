💻 Descrição do Projeto
Este projeto é a refatoração do sistema SysBiblio em Java, 
organizado em camadas com separação de responsabilidades.

Arquivos: Input.java, Livro.java, LivroService.java e Main.java.

Funcionalidades: cadastrar, listar, pesquisar (por título, autor 
ou ano), remover, editar e ordenar livros por ano de publicação.
